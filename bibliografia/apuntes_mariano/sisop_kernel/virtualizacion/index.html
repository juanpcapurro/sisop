Este sencillo programa, escrito en C, lo único que hace es ejecutar en forma infinita la impresión por pantalla del string que se le pasa como primer argumento. Al ejecutarlo, lo único que realizará será esa tarea en forma infinita… total… tiene toda la Unidad Central de Procesamiento para él.

#include <stdio.h>
#include <stdlib.h>
#include "../common.h"

int main(int argc, char *argv[])
{
    if (argc != 2) {
	fprintf(stderr, "usage: cpu <string>\n");
	exit(1);
    }
    char *str = argv[1];

    while (1) {
	printf("%s\n", str);
	Spin(1);
    }
    return 0;
}
